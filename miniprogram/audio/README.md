# 小程序音频文件说明

## 文件位置
请将以下音频文件放置在此目录中：

```
miniprogram/audio/
├── morning.mp3    # 晨曦云英 - 清晨冥想引导
├── noon.mp3       # 午间云英 - 正午冥想引导
├── evening.mp3    # 暮色云英 - 傍晚冥想引导
├── forest.mp3     # 林间鸟鸣 - 自然之声
├── stream.mp3     # 溪水潺潺 - 流水声
└── bell.mp3       # 古刹钟声 - 禅钟悠扬
```

## 音频规格建议
- 格式: MP3 (推荐) 或 AAC
- 比特率: 128kbps 或更高
- 采样率: 44100Hz
- 时长: 建议至少 7 分钟（可循环播放）

## 音乐分类

### 云英引导系列（3段）
适合需要语音引导的冥想场景，配合呼吸节奏。

### 自然之声（3段）
- **林间鸟鸣**：森林环境音，适合需要自然氛围的冥想
- **溪水潺潺**：流水声，适合需要洗涤、净化感的冥想
- **古刹钟声**：禅钟悠扬，适合需要空灵、澄澈感的冥想
