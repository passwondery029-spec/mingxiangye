# 冥想夜 🌸

一款冥想疗愈应用，用户输入执念后通过 7 分钟冥想，AI 生成专属画作和诗句。

## 多端版本

本项目提供两个版本：

| 版本 | 目录 | 技术栈 | 适用场景 |
|------|------|--------|----------|
| Web 版 | `/` (根目录) | React + Vite + Tailwind | 网页、H5 |
| 小程序版 | `/miniprogram` | 微信小程序原生 | 微信小程序 |

## 功能特性

- **执念输入** - 用户输入心中的执念或烦恼
- **情绪炼金** - AI（ARK 模型）根据执念生成疗愈诗句
- **智能配图** - 根据用户输入关键词匹配 100 张精美绘画风格图片
- **冥想体验** - 7 分钟引导冥想
- **资产沉淀** - 历史记录保存

## 图片库系统

### 100 张精选绘画风格图片

| 分类 | 数量 | 风格 |
|------|------|------|
| 山水意境 | 15 张 | 水墨、写意、光影 |
| 水之意象 | 15 张 | 静水、流动、海洋 |
| 花草植物 | 20 张 | 莲花、樱花、竹梅 |
| 天空云霞 | 15 张 | 日出日落、星空彩虹 |
| 禅意空间 | 15 张 | 茶室、庭院、枯山水 |
| 季节更替 | 10 张 | 春夏秋冬四时 |
| 动物生灵 | 10 张 | 蝴蝶、飞鸟、鱼鹿 |

### 智能匹配系统

每张图片包含：
- **主题标签** - 如：山、水、花、月等
- **情感标签** - 如：宁静、希望、自由等
- **元素标签** - 如：mountain, lotus, moon 等

系统会分析用户输入，匹配最合适的图片：
```
用户输入: "我最近工作压力很大，感到焦虑"
↓
匹配关键词: 工作、压力、焦虑
↓
匹配情感: 宁静、放松、释然
↓
推荐图片: 禅意空间类 或 山水意境类
```

## 环境变量配置

### 需要配置的环境变量

| 变量名 | 说明 | 获取方式 |
|--------|------|----------|
| `ARK_API_KEY` | ARK API 密钥 | 火山引擎控制台 |
| `ARK_MODEL_ID` | ARK 模型 ID | 火山引擎控制台 |

### 获取 ARK API Key

1. 访问 [火山引擎](https://www.volcengine.com/)
2. 进入 [ARK 控制台](https://console.volcengine.com/ark)
3. 创建 API Key 和模型接入点

### Vercel 配置步骤

1. **Settings** → **Environment Variables**
2. 添加：
   ```
   ARK_API_KEY = 你的API密钥
   ARK_MODEL_ID = ep-xxxxxxxxxxxx
   ```
3. **Save** 后 **Redeploy**

## API 接口

### POST /api/meditation/start

生成疗愈内容。

**响应：**
```json
{
  "status": "completed",
  "result": {
    "title": "心境",
    "poem": "心若止水，\n万物皆空。",
    "imageBase64": "https://images.unsplash.com/...",
    "matchedTags": ["山", "远山", "高山"],
    "matchedEmotions": ["坚定", "向上", "力量"]
  }
}
```

### GET /api/images/preview?keyword=焦虑

预览图片库或测试关键词匹配。

## 本地开发

```bash
# 安装依赖
pnpm install

# 配置环境变量
echo "ARK_API_KEY=your_key" > .env
echo "ARK_MODEL_ID=ep-xxx" >> .env

# 启动
pnpm dev
```
