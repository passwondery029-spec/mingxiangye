## 音频规格
- 格式: MP3
- 采样率: 24000Hz
- 语音: Jitangnv (`zh_female_jitangnv_saturn_bigtts`)
- 语速: -12（稍慢）
- 时长: 每段7分钟
- 背景: 低频嗡鸣（55Hz + 82Hz + 粉红噪音），全程持续

## 生成方式

### 引导音频（TTS + 背景音混合）
```bash
cd /workspace/projects/vibecoding-production/mingxiangye
npx tsx scripts/generate-guided-audio-v3.ts
```

### 自然音（ffmpeg合成）
```bash
bash scripts/generate-nature-audio.sh
```
